# 房價中位數儀表板

這是一個展示台灣各縣市房價中位數資料的互動式儀表板專案。

## 功能特色

- 📊 互動式圖表展示各縣市房價中位數趨勢
- 📅 時間軸事件標記（政策法規、重大事件等）
- 🗺️ 多縣市資料比較
- 📱 響應式設計，支援各種裝置

## 檔案說明

- `dashboard_standalone.html` - 主要儀表板頁面
- `1020_dashboard.html`, `1022_dashboard.html`, `1027_dashboard2.html`, `1013.html` - 各版本儀表板
- `dashboard_data.js` - 資料載入模組
- `json` - 資料檔案（包含房價中位數和事件資料）

## 使用方式

1. 直接在瀏覽器開啟任一 HTML 檔案
2. 或透過 GitHub Pages 線上瀏覽

## 資料來源

資料包含：
- 各縣市房價中位數（按季度）
- 政策法規事件時間軸
- 重大事件標記

## 技術架構

- 純前端實作（HTML + JavaScript）
- 使用 Chart.js 繪製圖表
- 資料以 JSON 格式儲存
